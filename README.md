アプリ名：もちもの帳 (Mochimono App)
〜地域の善意を見える化し、支え合いを加速させる〜

🌟 プロジェクトの概要

本アプリは、栃木県宇都宮市を拠点とする「地域支え合い研究会」によって企画・開発された試作品です。
災害時や日常の困りごとにおいて、「誰が・どこに・何を」持っているのかを可視化し、地域内でのスムーズな物資提供と助け合いを実現することを目的にしています。

🎯 開発の目的

社会には多くの支援物資や道具が眠っていますが、その保管場所は個人の記憶に頼っているのが現状です。

保管場所の見える化: 写真を撮るだけで位置情報を自動記録し、地図上にマッピングします。

善意のネットワーク: 「貸せるよ」「あげるよ」という個人の善意を繋ぎます。

即時性の確保: 必要な時に、一番近くにある物資へアクセスできる環境を整えます。

🤝 運用ポリシー（大切なお願い）

このアプリは、純粋な善意によって成り立つプラットフォームです。

口コミ限定の普及: 信頼できる関係性の中で利用していただくため、不特定多数への広告は行わず、顔の見える範囲でのご紹介のみで広げていく予定です。

非営利目的: 地域コミュニティ（PTAや自治会など）での助け合いを主眼に置いています。

✨ 主な機能

位置情報付き写真登録: 撮影した写真のEXIFデータから、保管場所を自動的にピン立てします。

エモーショナルなUI: 80年代SFアニメやサイバーパンクの美学を取り入れた、使うのが楽しくなるデザインを目指しています。

シンプル操作: 複雑な設定なしに、直感的に「物」と「場所」を紐付けられます。

🛠 セットアップ方法 (開発者向け)

リポジトリをクローン

npm install

.env ファイルを作成し、Google Maps APIキーを設定

npm run dev で起動
